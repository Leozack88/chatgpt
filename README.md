# QA Bug Report Smart Builder

Мини-проект для QA-портфолио: одностраничное приложение, которое помогает быстро оформить баг-репорт.

## Что умеет

- Генерирует баг-репорт в Markdown.
- Генерирует JSON-версию отчёта.
- Автоматически предлагает Severity и Priority по тексту бага.
- Считает QA Quality Score — насколько хорошо заполнен отчёт.
- Копирует отчёт в буфер обмена.
- Скачивает отчёт как `bug-report.md`.
- Сохраняет баг-репорты в LocalStorage браузера.
- Показывает историю сохранённых багов.
- Позволяет открыть старый отчёт обратно в форму.
- Позволяет удалить один отчёт или всю историю.
- Экспортирует всю историю в `bug-report-history.json`.

## Как запустить

Открой файл `index.html` в браузере.

## Как пользоваться

1. Заполни данные бага.
2. Нажми `Сгенерировать отчёт`.
3. Проверь Markdown или JSON.
4. Нажми `Сохранить в историю`.
5. Внизу страницы появится карточка сохранённого баг-репорта.
